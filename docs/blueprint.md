# **App Name**: Hovidcoverage

## Core Features:

- Provider Information: Capture MD name (First Name and Last name), Specialty, and Clinic details.
- Coverage Type: Select type of coverage (Inbase and outbase via dropdown).
- Coverage Date: Record the date of coverage.
- Photo Upload: Upload photos related to coverage events.
- Signature Capture: Capture signatures using a signature panel.
- Offline Coverage: Offline data capture and storage; data automatically synchronizes when internet is restored.
- Master List & Call Summary: Consolidated view of all coverage events with filtering and sorting options.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and professionalism in medical settings.
- Background color: Light gray (#F0F2F5) to reduce eye strain and maintain a clean, professional appearance.
- Accent color: Teal (#009688) to draw attention to key actions.
- Body font: 'PT Sans' (sans-serif) for clarity and readability in data entry fields and tables.
- Headline font: 'Space Grotesk' (sans-serif) for a modern look in headers, labels and titles.
- Use simple, clear icons to represent actions and status (e.g., upload, save, sync).
- Tab-based navigation for clear separation of features (Coverage, Offline Coverage, Master List, Call Summary).
- Subtle animations for feedback during data saving and synchronization processes.
